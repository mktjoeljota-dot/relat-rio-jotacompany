# Projeto: Dashboard ATRE

## Sobre
Dashboard de relatórios da ATrinca Real Estate (educação imobiliária).
Arquivo principal: index.html.html (HTML estático com Chart.js).

## Stack
- HTML/CSS/JS puro (sem framework)
- Chart.js 4.4.1 para gráficos
- Fonte: Geist (Google Fonts)
- Deploy: Vercel (auto-deploy via push na main)

## Identidade visual
- Tema dark
- Cores principais: verde (#10b981), âmbar (#f59e0b), vermelho (#ef4444)

## Padrões
- Porcentagens nos gráficos sempre fixas (não em hover)
- Commits em português, prefixo "feat:", "fix:", "style:" etc

## Histórico de alterações importantes
- Adicionadas porcentagens fixas
